# bubchat

Basic websocket chatroom TUI

## Run locally
First Clone this repo:
```
$ git clone https://github.com/harrysharma1/bubchat
```
Then either:
### Build
```
$ go build -o bubchat
$ ./bubchat --version
$ bubchat (v0.0.1)   
```
### Run
```
$ go run main.go --version
$ bubchat (v0.0.1)
```
## Built with

- [gorilla/websocket](https://github.com/gorilla/websocket/tree/main)
- [charmbracelet/bubbletea](https://github.com/charmbracelet/bubbletea)
